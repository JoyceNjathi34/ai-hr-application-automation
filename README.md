# AI HR Application Automation (n8n)

This project is an automated HR workflow built using n8n.

It automates the process of receiving and organizing job applications.

## Features

• Captures job applications automatically  
• Extracts candidate information  
• Stores applicant data in a structured format  
• Sends automatic responses to applicants  
• Helps HR teams manage applications efficiently  

## Workflow

1. Application received
2. Candidate information extracted
3. Data stored in database/CRM
4. Automated response sent

## Tools Used

- n8n
- AI model
- Google Sheets / CRM
- Email automation

## How to Use

Import the workflow JSON file into n8n and configure the credentials.
